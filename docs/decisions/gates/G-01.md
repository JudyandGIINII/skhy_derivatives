# G-01 — 보통주↔ADR 전환 운영상태

| 필드 | 내용 |
| --- | --- |
| 상태 | `UNKNOWN` |
| 질문 | Citi/KSD 및 실제 사용 브로커의 보통주↔ADR 전환 개시·방향·최소수량·비용·처리기간 |
| 기능 범위 | H2 신규 페어 진입 게이트 (`ConversionStatus`) |
| 공식 URL/문서 버전 | (미확인) |
| 계정·시장 범위 | (미확인) |
| 확인시각 | (미확인) |
| 원본 증거 checksum | (미확인) |
| 담당 provider | Citi(예탁기관), 실제 사용 브로커 |
| 결론 | (미확인) |

| 미확인 시 기본동작 | `ConversionStatus=UNKNOWN`, H2 신규 진입 차단 |

## 이력

- (초기 등록) 상태 `UNKNOWN`으로 시작. Phase 2 H2 실행 가능 실험 전 해소 필요.

언론·SNS 단독 보도는 이 게이트를 `CONFIRMED`로 바꾸지 못한다. Citi/KSD 공식 공지와
실제 사용 브로커의 접수 가능 화면 또는 서면 답변만 증거로 인정한다.
