## 이력

- **2026-07-18 (IN_REVIEW)**: KRX, KIS, Toss 3개 주요 공급자 이용약관 라이선스 URL을 확보하고 1차 예비 조사를 반영. 상세 조사 기록은 [evidence/G-06-data-terms.md](evidence/G-06-data-terms.md)에 기록됨. 조사 출처는 웹검색 1차 조사이며 공식 약관 전문 미대조(증거 문서 상단 주의 참고).
- **2026-07-18 (CONFIRMED)**: 시스템 소유자가 본 시스템을 개인용 단일 호스트 연구 시스템으로 규정하고, 개인 연구 목적 범위에서 데이터 수집·로컬 저장을 제약 없이 진행하기로 결정. 상태를 `CONFIRMED`로 전환. **주의**: 이 결정은 공식 약관 전문 검증에 근거한 법적 판단이 아니라 소유자 본인의 개인용 사용 판단·위험 인수다. 외부 배포·상업적 이용으로 용도가 바뀌면 아래 항목을 재확인하고 게이트를 재검토해야 한다.

### 용도 변경(외부 배포·상업적 이용) 시 재확인이 필요한 항목:
- [ ] **로컬 영구/장기 저장 허용 전문 확인**: KRX/KIS/Toss 상세 약관 전문(로그인 후)에서 장기 저장·축적 허용 여부 검토.
- [ ] **호출 한도 준수 확인**: 백필 파이프라인의 API 트래픽 제어(TPS, 일일 한도)가 각 공급자 제한 내로 수렴하는지 확인.
- [ ] **외부 유출 차단 검증**: 원천/정규화 데이터가 외부 공개 저장소로 푸시되거나 타인에게 공유되지 않는지 재검증.
- [ ] **재배포 조항 확인**: 각 공급자 약관의 재배포·제3자 제공 금지 조항을 대조.

- (초기 등록) 상태 `UNKNOWN`. **어떤 실데이터의 지속 수집보다도 먼저** 해소해야 하는 최우선 게이트였음 (`implementation_plan.md` 7.2 착수 순서 1번).
