# G-06 증거: 공급자별 데이터 이용약관 조사 결과

이 문서는 Phase 1 범위에 관여하는 3개 핵심 공급자(KRX, KIS, Toss)의 오픈 API 이용약관을 조사한 증거 자료입니다.

> **⚠️ 출처·신뢰도 주의**: 아래 "[웹검색 1차 조사 — 약관 전문 미대조]" 표기는 **웹 검색 기반 1차 조사** 결과이며, 로그인 후 접근하는 공식 약관 전문(또는 약관 PDF)을 직접 대조해 검증한 것이 **아닙니다**. LLM 조사의 해석이 포함되어 있으므로 법적 효력이 있는 결론으로 사용하기 전에 사용자가 반드시 각 공급자 약관 전문을 직접 확인해야 합니다. 특히 호출 한도(TPS·일일 한도) 수치는 계정·서비스별로 다를 수 있으므로 실제 값 확인 없이 백필을 돌리면 계정 차단 위험이 있습니다. 보수적 결론(재배포 전면 금지, 로컬 최소 보관)은 확인 여부와 무관하게 안전한 기본값으로 채택합니다.

## 1. 공급자별 이용약관 요약

### 1) KRX (한국거래소) Open API
*   **공식 라이선스 URL**: [https://openapi.krx.co.kr/contents/OPP/INFO/service/OPPINFO004.cmd](https://openapi.krx.co.kr/contents/OPP/INFO/service/OPPINFO004.cmd)
*   **(a) 데이터 저장/캐싱 허용 여부 및 기간**: **[미확인, 사용자 확인 필요]**
    *   *조사 내용*: 개인이 비상업적인 연구 및 분석 목적으로 데이터를 로컬(CSV 등)에 보관하는 것은 통상 허용되는 범위로 알려져 있으나, 영구 저장 또는 장기 캐싱에 대한 명시적인 보관 허용 기간은 공식 약관 전문에서 사용자가 직접 재확인해야 합니다.
*   **(b) 자동수집 허용 여부 및 호출 한도**: **[웹검색 1차 조사 — 약관 전문 미대조]**
    *   *조사 내용*: 승인된 API 인증키를 사용한 비상업적 목적의 프로그램 조회(자동 수집)를 공식 허용합니다. 단, 서비스별 일일 호출 횟수 한도(예: 10,000회 등)가 있으며 초과 시 차단될 수 있습니다.
*   **(c) 재배포·제3자 제공 가능 범위**: **[웹검색 1차 조사 — 약관 전문 미대조]**
    *   *조사 내용*: 상업적 이용 및 제3자 제공, 외부 재배포는 원칙적으로 엄격히 금지됩니다.

### 2) KIS (한국투자증권) Open API
*   **공식 라이선스 URL**: [https://apiportal.koreainvestment.com/intro](https://apiportal.koreainvestment.com/intro)
*   **(a) 데이터 저장/캐싱 허용 여부 및 기간**: **[미확인, 사용자 확인 필요]**
    *   *조사 내용*: 자기 자산 관리 및 개인 투자 분석을 목적으로 로컬 스토리지에 캐싱하는 것은 관행적으로 행해지나, DB에 원천(Raw) 데이터를 영구 또는 수년간 적재하는 행위에 대해 규제/제한 조항이 있는지 여부는 공식 약관 전문을 통해 검토해야 합니다.
*   **(b) 자동수집 허용 여부 및 호출 한도**: **[웹검색 1차 조사 — 약관 전문 미대조]**
    *   *조사 내용*: API 키를 발급받은 개인의 자기 자산 매매 및 시세 조회를 위한 프로그램 구동을 공식 지원합니다. 단, 초당 호출 제한(TPS) 및 API 요청 과부하 시 차단 규정이 적용됩니다.
*   **(c) 재배포·제3자 제공 가능 범위**: **[웹검색 1차 조사 — 약관 전문 미대조]**
    *   *조사 내용*: 제공되는 시세 및 데이터는 계좌 소유주 본인의 투자 목적 외 제3자 제공이나 외부 재배포가 원칙적으로 전면 금지됩니다. (외부 서비스 표출 시 코스콤 등 거래소와 별도 계약 필요)

### 3) Toss (토스증권) Open API
*   **공식 라이선스 URL**: [https://openapi.tossinvest.com/openapi-docs/overview.md](https://openapi.tossinvest.com/openapi-docs/overview.md)
*   **(a) 데이터 저장/캐싱 허용 여부 및 기간**: **[미확인, 사용자 확인 필요]**
    *   *조사 내용*: 연구 목적으로 백테스팅용 시세 DB를 구축(120거래일 이상의 데이터 축적)하는 행위의 허용 범위와 기간 상한에 대해서는 '오픈 API 서비스 이용약관' 전문을 직접 확인해야 합니다.
*   **(b) 자동수집 허용 여부 및 호출 한도**: **[웹검색 1차 조사 — 약관 전문 미대조]**
    *   *조사 내용*: 공식 오픈 API 키를 활용한 프로그램 조회 및 자동 주문을 지원합니다. 단, 공식 외 웹앱 내부 API 우회, 비공식 경로를 통한 스크래핑/CLI 접근은 약관 위반으로 계좌 제재 및 차단 조치 대상입니다.
*   **(c) 재배포·제3자 제공 가능 범위**: **[웹검색 1차 조사 — 약관 전문 미대조]**
    *   *조사 내용*: API 제공 데이터 및 문서 저작권은 토스증권에 귀속되며, 상업적 활용 및 무단 재배포/가공 후 배포는 엄격히 금지됩니다.

## 2. 권장 기본 조치 (보수적 접근)

공급자 약관 조사 결과, 개인 연구 용도의 로컬 활용은 통상 가능하나 재배포는 전면 금지되어 있으므로 본 시스템은 다음의 보수적 원칙을 고수해야 합니다.
1. **외부 재배포 차단**: 수집한 모든 데이터(원천 raw, 정규화 normalized, lineage 등)는 단일 호스트 내부 가상환경 밖으로 유출하거나 재배포하지 않습니다.
2. **저장기간 상한 수립**: 연구에 필요한 백필 파이프라인 정합성 검증 기간(최소 120거래일)을 충족하는 범위 내에서 데이터를 보관하되, 불필요한 장기 적재를 막기 위해 **원천(Raw) 데이터 보관 기한을 최대 1년(365일)**으로 제한하고, 초과된 오래된 로그 및 Raw 데이터는 자동 삭제 정책을 적용하는 것을 권장합니다.
